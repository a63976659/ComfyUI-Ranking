# ComfyUI-Ranking

> ComfyUI 精选社区 —— 让创作有价值，让分享有回报

一个深度集成于 ComfyUI 的内容社区插件，支持工具发布、应用分享、资源推荐、任务委托等功能，打造 ComfyUI 生态的价值闭环。

---

## 🎯 项目愿景

1. **让创作变现** —— 工具/工作流可直接定价销售，创作者持续获得收益
2. **让优质可见** —— 多维度榜单排序，好内容更容易被发现和获取
3. **让分享有回报** —— 建立健康的赞赏机制，无偿分享也能获得支持
4. **让生态繁荣** —— 正向循环驱动社区持续进步

---

## ✨ 核心功能

### 📊 六大榜单

| 榜单 | 说明 | 动画效果 |
|------|------|----------|
| **工具榜** | 发布 ComfyUI 节点插件，支持 GitHub 私有库绑定防盗 | 🌊 瀑布式载入 |
| **应用榜** | 分享工作流 JSON，一键下载并导入工作区 | 🌊 瀑布式载入 |
| **推荐榜** | 类小红书的图文内容，封面图 + 详情图展示 | 🌊 瀑布式载入 |
| **讨论区** | 社区交流，支持富文本发帖与评论树 | 🕳️ 深渊汇聚 |
| **任务榜** | 发布外包需求，支持订金托管与尾款结算 | ⚡ 数据流滑入 |
| **创作者榜** | 优质创作者排行，展示贡献与粉丝数据 | 🌀 扇形展开 |

### 🚀 极简体验

- **一键安装** —— 静默安装插件到 `custom_nodes`，云端内存解压，无需手动 `git clone`
- **一键加载** —— 下载工作流后直接添加到工作区，已购内容永久有效
- **自动更新** —— 定时检测 GitHub 版本，发现新版一键覆盖更新

### 💬 社交互动

- **点赞收藏** —— 标记喜欢的内容，方便后续查找
- **评论系统** —— 支持嵌套评论树，防崩溃设计
- **私信聊天** —— 即时通讯，消息实时推送
- **通知中心** —— 点赞、回复、购买等全局消息提醒

### 💰 价值体系

- **定价自由** —— 自由设置积分价格（1积分=1元），或设为免费
- **充值便捷** —— 支付宝扫码秒充值
- **账目分离** —— 销售收入与打赏收入双轨独立
- **安全提现** —— 阶梯手续费，提现至支付宝

### 🔒 安全保障

- **内容保护** —— 付费内容云端永久记录，一次购买终身有效
- **私有库防盗** —— 支持 GitHub 私有库密钥绑定
- **账号安全** —— 基于邮箱的安全验证与风控
- **图片审核** —— 多云平台智能审核，过滤违规内容

## 📦 安装方式

### 方式一：Git Clone（推荐）

```bash
# 进入 ComfyUI 插件目录
cd ComfyUI/custom_nodes

# 克隆仓库
git clone https://github.com/a63976659/ComfyUI-Ranking.git

# 重启 ComfyUI
```

### 方式二：手动安装

1. 下载仓库 ZIP 文件
2. 解压到 `ComfyUI/custom_nodes/ComfyUI-Ranking`
3. 重启 ComfyUI

### ⚠️ 安装注意

- 安装失败时 `custom_nodes` 中可能存在空的 `ComfyUI-Ranking` 文件夹
- 删除该空文件夹后重新安装即可

## 🎮 使用指南

### 创作者

1. 安装插件并注册社区账号
2. 发布原创工具（可绑定私有库防盗）或工作流应用
3. 自由设置定价或免费开源
4. 将销售收益提现至支付宝

### 分享者

1. 安装插件并注册账号
2. 搬运推荐优质资源，或无偿分享好用的工作流
3. 积累粉丝与口碑
4. 获得粉丝打赏并提现

### 普通用户

1. 浏览多维度榜单（最新/下载量/收藏量）
2. 一键安装优质工具提升生产力
3. 导入适合自己的工作流
4. 通过真实评价和购买量避坑无效资源

## 🏗️ 技术架构

```
ComfyUI-Ranking/
├── 前端页面/
│   ├── auth/          # 用户认证组件
│   ├── components/    # 通用UI组件
│   ├── core/          # 核心引擎（状态管理、网络请求）
│   ├── market/        # 市场功能（发布、详情、钱包）
│   ├── profile/       # 个人中心
│   └── social/        # 社交功能（私信、评论、通知）
├── 云端Space代码/       # 后端 FastAPI 服务
├── api_app.py         # 本地 API 入口
├── api_cache.py       # 缓存管理
└── api_tool.py        # 工具函数
```

### 技术栈

- **前端**: 原生 JavaScript + CSS3 动画 + Web Audio API
- **后端**: Python FastAPI + SQLite/PostgreSQL
- **部署**: Hugging Face Spaces (Docker)
