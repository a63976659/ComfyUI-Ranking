# ComfyUI-Ranking
ComfyUI中的应用排行榜工具



ComfyUI 社区精选

该插件不仅彻底打通了本地与云端的数据壁垒，更细化了资源分发与数据追踪的颗粒度，形成了一个完美的去中心化创作者经济生态。

一、 核心设计理念 (Design Philosophy)
原生与无感融合 (Seamless Integration)：
放弃破坏沉浸感的全局大弹窗，完美融入 ComfyUI 标准的侧边栏机制 。引入应用级别的“内联路由”机制，让浏览、聊天、发布、个人设置等所有操作都在侧边栏内部平滑无缝切换，保护用户搭建工作流时的“心流”。

零依赖与极简前端：
前端完全基于纯原生 JavaScript 构建。这确保了插件拥有极致的加载速度，且永远不会与 ComfyUI 底层或其他复杂第三方插件产生依赖冲突。

数据护城河与隐私至上：

云端隐私：云端数据库托管于 云端 保密仓库，杜绝外部机器人的恶意爬取。

端侧隐私：赋予用户绝对的数据控制权，用户可自由隐藏自己的点赞、关注、收藏和下载轨迹。

创作者生态闭环：
提供“多维度发布 -> 一键分发/路由 -> 精准使用量追踪 -> 互动(评论/私信) -> 数据分析(三轨折线图) -> 粉丝沉淀”的完整生命周期，最大化激发社区创造力。

二、 核心功能特点 (Key Features)
🔐 账号与安全中心

支持账号注册、自动登录、云端认证。

找回密码支持“手机号/邮箱”的智能正则识别与后端动态处理。

🧰 全矩阵资源发布引擎

多栖发布：支持发布原创工具 (Tool)、原创应用 (App)，以及强大的推荐榜单 (Recommend)。

推荐子类细化：推荐榜单细分为 recommend_tool (作为工具安装)、recommend_app (作为工作流加载)、recommend_link (纯外链跳转) 三种形态。

大视野编辑：发布页面的详细说明框扩容高度，并全线放开封面图上传权限。

💬 全局社区互动网络

通知中心：顶栏铃铛汇总点赞、收藏、关注、留言等动态，支持未读红点和本地/云端分离的数据清理。

即时通讯：内置左右分栏的私信系统，支持按联系人分发消息，超 7 天已读消息自动清理。

留言板系统：支持对资源、对创作者主页的嵌套式软删除回复（盖楼）。

📊 创作者高阶数据看板

三轨数据大盘：自动按月绘制包含“原创工具”、“原创应用”、“推荐获取”的三根高精度折线图（最低刻度 1 级，拒绝半数显示）。

三、 核心功能关联与数据流转 (Feature Interactions)
在这个系统中，数据的流转是高度解耦且实时联动的（牵一发而动全身）：

“泛获取”的生态联动：

用户点击“立即获取” -> 前端根据类型分发（如 工具 走 Git 克隆，应用 走 JSON 热加载，第三方链接 走新标签页）。

-> 随后，所有类型的获取都会静默加载 提示页面切换流畅度。

-> 云端：将该资源的动态数据记录。

-> 图表联动：云端 会聚合该作者名下所有的 工具、应用 和 推荐 数据。当用户访问主页时，增长数据 会实时渲染出三根同步抬升的真实趋势曲线。

“修改与删除”的防级联破坏：

创作者修改发布内容 -> 前端进入“修改”回显旧数据 -> 保存后云端执行 覆盖 更新（保留原有的点赞、下载、评论等交互数据完全不变）。

创作者删除内容 -> 云端执行  移除资源，触发级联清理，去 仓库 中同步销毁该资源下的所有评论记录，防止产生“幽灵评论数据”。

“隐私设置”的前后端闭环：

用户在设置中勾选“隐藏我的关注列表” -> 云端更新 用户数据显示权限。

当其他人访问其主页时 -> 前端拉取数据，识别到 关闭展示 -> 强行阻断关注列表  的渲染，从 云端 层面杜绝数据泄露。

四、 技术亮点与实现细节 (Technical Highlights)
时间序列的云端下沉计算
抛弃前端生成静态伪造时间轴的做法。确保所有用户看到的时间切片绝对一致。

基于 界面 的切换展示 (Inline Routing)
彻底消灭了传统弹窗导致的 Z-index 穿透和遮挡地狱。实现了极其流畅的 SPA 体验。

高性能图片沙盒算法 (Image Sandbox)
针对创作者上传的封面图，手工实现了高性能的图片沙盒查看器。让高清节点连线图的细节分毫毕现。
